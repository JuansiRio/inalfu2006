# SEO quick checklist

Use after changes or as an audit pass.

- [ ] `<html lang="...">` matches page language
- [ ] Unique, accurate `<title>` and meta description
- [ ] Canonical URL if duplicates or syndication exist
- [ ] Open Graph + Twitter Card tags; `og:image` is absolute URL in production
- [ ] JSON-LD only for truthful, on-page content
- [ ] One clear `<h1>`; headings in order
- [ ] Meaningful `alt` on informative images
- [ ] Important images have dimensions or stable aspect ratio
- [ ] Internal links use descriptive anchor text
- [ ] `robots.txt` / `sitemap.xml` updated when site structure or domain changes
