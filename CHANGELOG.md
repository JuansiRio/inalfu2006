## Changelog

Todos los cambios relevantes de este proyecto se documentarán en este archivo.

## [1.0.0] - 2026-03-09

### Added

- Estructura base del proyecto con HTML, TailwindCSS y JavaScript Vanilla.
- Landing page completa para VitalPaws con secciones de hero, servicios, beneficios, testimonios, contacto y footer.
- Menú responsive para navegación móvil.
- Animaciones de entrada al hacer scroll usando Intersection Observer.
- Carrusel de testimonios con navegación manual y autoplay.
- Validación básica del formulario de contacto.
- Scripts de desarrollo y build para compilación de estilos con TailwindCSS.
