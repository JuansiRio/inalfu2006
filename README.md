## 10X Builder Code

Landing page para veterinaria construida con **HTML5**, **TailwindCSS** y **JavaScript Vanilla**.

El objetivo del proyecto es ofrecer una experiencia visual moderna, confiable y orientada a conversión para una clínica veterinaria ficticia llamada **VitalPaws**.

## Características

- Hero con propuesta de valor y CTAs (agendar, WhatsApp, llamada)
- Sección de servicios, beneficios, testimonios y contacto
- Navegación responsive con menú móvil
- Animaciones de entrada al hacer scroll (Intersection Observer)
- Carrusel de testimonios con controles y autoplay
- Validación básica de formulario de contacto

## Stack

- **HTML5**
- **TailwindCSS 3**
- **JavaScript (Vanilla)**

## Requisitos

- Node.js 18+ (recomendado)
- npm 9+ (recomendado)

## Instalación

```bash
npm install
```

## Scripts disponibles

- `npm run dev`: compila Tailwind en modo watch
- `npm run build`: compila Tailwind en modo minificado

## Estructura del proyecto

```text
.
├── docs/
├── public/
│   └── assets/
│       └── css/
├── src/
│   └── styles/
├── index.html
├── script.js
└── package.json
```

## Desarrollo local

1. Instala dependencias: `npm install`
2. Corre Tailwind en modo desarrollo: `npm run dev`
3. Abre `index.html` en tu navegador
